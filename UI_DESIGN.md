# AI知识库 UI/UX 专业设计方案

## 设计目标
- 专业、简洁、高效的半导体工厂知识管理系统界面
- 符合B端产品设计语言
- 突出AI能力的智能化体验

## 设计系统

### 色彩体系
```
主色调: #1677FF (Ant Design Blue)
辅助色: #52C41A (Success) / #FAAD14 (Warning) / #F5222D (Error)
中性色: #262626 (标题) / #595959 (正文) / #8C8C8C (次要) / #BFBFBF (禁用)
背景色: #F0F2F5 (页面) / #FFFFFF (卡片)
```

### 字体规范
```
标题: 24px/32px/40px PingFang SC Medium
正文: 14px PingFang SC Regular
代码: 12px Monaco / Consolas
```

### 布局规范
```
侧边栏: 240px 固定宽度（可折叠至80px）
内容区: 自适应，最大1440px
卡片间距: 24px
内边距: 24px
```
