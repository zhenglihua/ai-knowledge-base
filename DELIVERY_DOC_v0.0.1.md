## 一、系统概述

### 1.1 项目背景

专为半导体工厂设计的智能知识管理系统，基于 RAG (Retrieval-Augmented Generation) 技术，实现文档智能检索与 AI 问答。

### 1.2 技术架构

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 后端框架 | FastAPI | 高性能 Python Web 框架 |
| 数据库 | SQLite | 轻量级关系数据库 |
| 向量存储 | Qdrant | 向量相似度检索 |
| Embedding | sentence-transformers | 多语言模型 paraphrase-multilingual-MiniLM-L12-v2 |
| 前端框架 | React 18 + TypeScript | 组件化开发 |
| UI 组件库 | Ant Design 5 | 企业级 React UI |
| 构建工具 | Vite | 快速的开发服务器和构建工具 |

## 三、API 接口文档

### 3.1 认证接口

| 方法 | 路径 | 描述 | 请求体 |
|------|------|------|--------|
| POST | `/api/auth/login` | 用户登录 | `{username, password}` |
| POST | `/api/auth/register` | 用户注册 | `{username, email, password}` |
| POST | `/api/auth/logout` | 用户登出 | - |

### 3.2 文档接口

| 方法 | 路径 | 描述 | 请求体/参数 |
|------|------|------|------------|
| GET | `/api/documents` | 获取文档列表 | `?keyword=&category=` |
| POST | `/api/documents/upload` | 上传文档 | `multipart/form-data` |
| DELETE | `/api/documents/{id}` | 删除文档 | - |

### 3.3 搜索接口

| 方法 | 路径 | 描述 | 请求体 |
|------|------|------|--------|
| POST | `/api/search` | 语义搜索 | `{keyword}` |

### 3.4 问答接口

| 方法 | 路径 | 描述 | 请求体 |
|------|------|------|--------|
| POST | `/api/chat` | AI问答 | `{query}` |

### 3.5 知识图谱接口

| 方法 | 路径 | 描述 | 请求体 |
|------|------|------|--------|
| GET | `/api/kg/graph` | 获取图谱数据 | - |
| POST | `/api/kg/entities` | 创建实体 | `{name, type, ...}` |
| PUT | `/api/kg/entities/{id}` | 更新实体 | `{name, ...}` |
| DELETE | `/api/kg/entities/{id}` | 删除实体 | - |
| POST | `/api/kg/relations` | 创建关系 | `{source, target, type}` |

### 3.6 用户管理接口

| 方法 | 路径 | 描述 | 请求体 |
|------|------|------|--------|
| GET | `/api/users` | 获取用户列表 | `?keyword=&status=` |
| POST | `/api/users` | 创建用户 | `{username, email, ...}` |
| PUT | `/api/users/{id}` | 更新用户 | `{...}` |
| DELETE | `/api/users/{id}` | 删除用户 | - |

## 四、部署指南

### 4.1 环境要求

| 项目 | 最低要求 | 推荐配置 |
|------|---------|---------|
| CPU | 2核 | 4核+ |
| 内存 | 4GB | 8GB+ |
| 磁盘 | 10GB | 20GB+ |
| Node.js | 16.0 | 18.0+ |
| Python | 3.9 | 3.10+ |

### 4.2 快速部署

#### 后端部署

```bash
# 1. 进入后端目录
cd ai-knowledge-base/mvp/backend

# 2. 安装依赖（使用国内镜像）
pip install -r requirements.cn.txt

# 3. 启动服务
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

#### 前端部署

```bash
# 1. 进入前端目录
cd ai-knowledge-base/mvp/frontend

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm start

# 或生产环境构建
npm run build
```

#### 一键启动（国内用户）

```bash
cd ai-knowledge-base/mvp
chmod +x start.cn.sh && ./start.cn.sh
```

### 4.3 Docker 部署

```bash
# 启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f backend
```

### 4.4 访问地址

| 服务 | 地址 | 说明 |
|------|------|------|
| 前端界面 | http://localhost:3000 | 用户访问入口 |
| 后端API | http://localhost:8000 | API服务地址 |
| API文档 | http://localhost:8000/docs | Swagger文档 |
| Qdrant | http://localhost:6333 | 向量数据库控制台 |

## 六、使用指南

### 6.1 登录系统

1. 访问 http://localhost:3000
2. 输入演示账号：`admin` / `123456`
3. 或使用注册账号登录

### 6.2 文档上传

1. 进入「文档管理」页面
2. 点击「上传文档」或拖拽文件
3. 选择 PDF、Word 或 TXT 文件
4. 系统自动解析并向量化

### 6.3 智能搜索

1. 进入「智能搜索」页面
2. 输入搜索关键词
3. 系统返回语义相关结果
4. 点击结果可查看详情

### 6.4 AI问答

1. 进入「AI问答」页面
2. 输入问题
3. 系统基于知识库生成回答
4. 答案标注参考来源

### 6.5 知识图谱

1. 进入「知识图谱」页面
2. 查看实体关系网络
3. 支持缩放、拖拽、点击查看详情
4. 进入「图谱管理」进行编辑
