# 清洗与干燥（Cleaning & Drying）技术手册

## 一、工艺概述

硅片清洗（Wafer Cleaning）是半导体制造中重复次数最多的工艺，贯穿整个IC制造流程。清洗的目的是去除硅片表面的颗粒（Particle）、有机物（Organic）、金属沾污（Metallic Contamination）和自然氧化层（Native Oxide），为下一步工艺提供洁净的表面。硅片清洗的质量直接影响器件的良率和可靠性。

硅片干燥（Wafer Drying）的目的是在清洗后去除硅片表面的水分，防止水痕（Watermark）、干燥颗粒（Dry Particle）和雾化（Haze）缺陷。

## 二、SC-1/RCA清洗

RCA清洗是半导体工业的标准湿法清洗工艺，由RCA公司（Radio Corporation of America）于1965年发明。

### 2.1 SC-1（标准清洗-1，NH₄OH/H₂O₂/H₂O）

目的：去除有机物和颗粒

标准工艺参数：
| 参数 | 典型值 | 说明 |
|------|--------|------|
| NH₄OH（25%） | 1:1:5~1:2:7（与H₂O₂:H₂O比例） | 标准比例1:1:5 |
| H₂O₂（30%） | 1份 | 氧化剂 |
| DI水 | 5-7份 | 稀释剂 |
| 温度 | 70-85°C | 最佳60-80°C |
| 时间 | 5-15分钟 | 根据污染程度 |
| 超声/Megasonic | 0.8-2 MHz | 颗粒去除 |
| 容器 | 石英或PTFE | 避免金属沾污 |

注：SC-1会轻微刻蚀硅表面（~1-2nm），不宜时间过长。

### 2.2 SC-2（标准清洗-2，HCl/H₂O₂/H₂O）

目的：去除金属沾污（Fe、Mg、Zn、Al等）

标准工艺参数：
| 参数 | 典型值 |
|------|--------|
| HCl（37%） | 1:1:6（与H₂O₂:H₂O比例） |
| H₂O₂（30%） | 1份 |
| DI水 | 6份 |
| 温度 | 70-85°C |
| 时间 | 5-15分钟 |

### 2.3 DHF清洗（稀释HF）

目的：去除自然氧化层和金属沾污（贵金属如Au、Ag等不溶于HF，但随氧化层去除）

标准工艺参数：
- HF（49%）：DI水 = 1:50~1:100（稀释HF，DHF）
- 或 HF（49%）：NH₄F = 1:6~1:10（缓冲HF，BHF/BOE）
- 温度：室温（20-25°C）
- 时间：10-60秒
- 注：DHF会去除1-5nm硅表面，SC-1前通常用DHF去除氧化层

### 2.4 SPM清洗（浓硫酸/双氧水）

目的：去除有机物和光刻胶残留（强力除有机物）

标准工艺参数：
- H₂SO₄（96-98%）：H₂O₂（30%）= 4:1~10:1
- 温度：100-150°C（热法）或室温（冷法）
- 时间：5-30分钟
- 注：SPM是强氧化性混合物，可将光刻胶彻底氧化去除

## 三、兆声波清洗（Megasonic Cleaning）

兆声波清洗利用高频（0.8-3 MHz）声波在液体中产生的空化效应去除亚微米级颗粒：

标准工艺参数：
- 频率：0.8-2 MHz（兆声波）
- 功率：50-500 W
- 温度：室温-60°C
- 时间：1-10分钟
- DI水流量：持续或间歇

优势：
- 比传统超声波（40kHz）更温和，损伤风险低
- 可去除<0.1μm的亚微米颗粒
- 对光刻胶图形无损伤

## 四、干燥工艺

### 4.1 旋转干燥（Spin Dryer）

最简单和常用的干燥方式：

标准参数：
- 转速：2000-5000 rpm
- 时间：30-120秒
- 温度：room temperature或加热（40-80°C）
- 注：可配合IPA干燥提高效果

### 4.2 IPA干燥（Isopropyl Alcohol）

利用IPA蒸汽冷凝带走水分：

标准IPA干燥参数：
- IPA纯度：>99.9%（电子级）
- N₂气流：5-20 L/min（干燥）
- IPA加热温度：60-80°C（加热IPA法）
- 时间：2-10分钟
- 方式：
  - 加热IPA蒸汽法（典型：Süss MicroTec设备）
  - 冷IPA置换法（室温IPA+DI水置换）

### 4.3 Marangoni干燥

利用表面张力梯度驱动液体从硅片表面撤离：

标准参数：
- DI水接触后，用IPA/N₂混合气体处理
- N₂流速：10-50 L/min
- IPA浓度梯度控制表面张力
- 温度：室温
- 效果：优于纯旋转干燥，水痕风险更低

### 4.4 超临界干燥（Supercritical CO₂ Drying）

用于MEMS等微结构释放工艺，防止毛细力破坏微结构：

标准参数：
- 压力：>1070 psi（>7.4 MPa，CO₂超临界点）
- 温度：35-50°C
- 介质：液态CO₂置换水后升压至超临界
- 注：成本高，仅用于特殊应用

## 五、先进清洗技术

### 5.1 干法清洗（Dry Cleaning）

不使用液体的清洗方式：

#### 等离子体清洗
- O₂等离子体：去除有机物/光刻胶
- H₂等离子体：去除金属氧化物（需无氧环境）
- Ar等离子体：物理轰击去除颗粒

#### UV/Ozone清洗
- UV灯（185nm+254nm）产生臭氧
- 去除有机污染：1-10分钟
- 温度：室温-100°C
- 应用：有机物污染清除

### 5.2 化学机械清洗（Chemical Mechanical Cleaning）

结合化学和机械作用，适用于顽固污染：

标准参数：
- 清洗液：SiO₂ slurry + 碱性清洗剂
- 压力：1-3 psi（低于CMP）
- 垫类型：软垫（PU或绒毛）
- 时间：1-5分钟

## 六、清洗质量控制

### 6.1 清洗质量检测

| 检测项 | 方法 | 规格 |
|------|------|------|
| 颗粒残留 | SURFSCAN或显微镜 | <10个/片（>0.2μm） |
| 金属沾污 | TXRF或VPD-ICPMS | <1×10¹⁰ atoms/cm² |
| 有机沾污 | XPS或FTIR | 无明显C-H峰 |
| 自然氧化层 | 椭偏仪或Ellipsometer | 1-3nm（正常范围） |
| 硅片表面粗糙度 | AFM（Ra） | <0.5nm（清洗后） |

### 6.2 金属沾污控制水平

关键金属沾污限值（先进制程）：
| 金属 | 规格（atoms/cm²） |
|------|-----------------|
| Fe | <1×10¹⁰ |
| Cu | <1×10¹⁰ |
| Ni | <1×10¹⁰ |
| Cr | <5×10⁹ |
| Na | <1×10¹⁰ |
| K | <1×10¹⁰ |

## 七、常见问题与解决方案

### 7.1 颗粒污染去除不干净

表现：清洗后片子上颗粒数量仍超标。

原因：
- 兆声波功率不足
- 清洗液浓度/温度偏差
- 颗粒附着过强（化学键合）
- 设备自身颗粒产生

解决方案：
- 检查兆声波发生器功率
- 验证清洗液配制（浓度、温度）
- 增加DHF处理破坏化学键合
- 排查清洗设备本身的颗粒来源

### 7.2 金属沾污超标

原因：
- 工艺设备金属沾污（腔室/管道/器具）
- 清洗液纯度不足
- 操作人员手套/工具污染

解决方案：
- 使用高纯度化学品（电子级）
- 设备定期酸洗钝化
- 加强操作规范（无金属工具）
- 使用HF最后处理去除表面污染层

### 7.3 水痕/雾化缺陷

表现：干燥后在片子上形成水渍或表面雾化。

原因：
- 干燥不充分
- DI水水质不佳（杂质离子）
- 旋转速度不足
- IPA纯度不足

解决方案：
- 改用IPA干燥或Marangoni干燥
- 检查DI水水质（电阻率>18 MΩ·cm）
- 提高旋转速度
- 使用高纯度IPA

## 九、安全注意事项

1. **强酸强碱安全**：HF（剧毒）、H₂SO₄（腐蚀）、HCl（腐蚀）、NH₄OH（刺激）操作需穿戴全防护装备
2. **HF特别警示**：HF穿透皮肤后与Ca²⁺结合导致组织坏死，需配备Calcium Gluconate凝胶急救包并知道急救程序
3. **H₂O₂安全**：高浓度双氧水为强氧化剂，遇有机物剧烈反应，储存和使用远离可燃物
4. **热法SPM安全**：热硫酸（100-150°C）烫伤风险极高，操作需特级防护
5. **IPA安全**：易燃（闪点12°C），远离明火和热源，配备防爆设施
6. **废液处理**：SC-1/SC-2废液含重金属和氨氮，需分类收集处理
7. **通风安全**：所有湿法操作需在排风柜/台内进行
