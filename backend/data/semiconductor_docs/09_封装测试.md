# 封装与测试（Packaging & Testing）技术手册

## 一、工艺概述

封装（Packaging）和测试（Testing）是半导体制造的后道工序，将晶圆上的芯片切割成单个芯片，进行电气连接、封装保护，并进行功能测试和可靠性筛选，最终交付给客户。封装不仅提供芯片的物理保护和对外电气接口，还直接影响芯片的散热性能、电气性能和可靠性。

封装技术从最早的双列直插封装（DIP）发展到今天先进的2.5D/3D封装和Chiplet技术，技术演进日新月异。

## 二、晶圆级工艺

### 2.1 晶圆切割（Wafer Dicing）

将晶圆上的芯片用金刚石刀片切割成单个芯片：

标准工艺参数：
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 刀片厚度 | 20-50μm（刀厚） | 取决于切割道宽度 |
| 刀片粒度 | 325-2000 mesh | 越细越光滑但效率低 |
| 主轴转速 | 30000-60000 rpm | 高速旋转 |
| 进刀速度 | 50-200 mm/s | 影响切割质量和效率 |
| 冷却水流速 | 2-8 L/min | 冷却和冲刷 |
| 切割深度 | 穿过晶圆+10-20μm | 确保完全切开 |

UV胶贴片（UV Tape）：
- 贴片UV胶厚度：5-20μm
- UV照射能量：200-1000 mJ/cm²
- 贴片压力：0.1-0.5 MPa

切割质量判定：
- 切口宽度：<刀片厚度+10μm
- 崩边（Chip-out）：<10μm
- 切口垂直度：>85°
- 刀片寿命：每500-2000刀更换

### 2.2 芯片贴装（Die Attach）

将切割后的芯片粘贴到引线框架或基板上：

#### 粘胶贴装（Epoxy Adhesive）

标准工艺参数：
- 胶水类型：导电胶（Ag填充）或绝缘胶
- 粘胶量：0.1-1 mg（点胶或印刷）
- 点胶高度：0.05-0.2 mm
- 贴装压力：0.1-0.5 MPa
- 固化温度：150-200°C
- 固化时间：30-120分钟（烤箱）或30-120秒（热压）

#### 共晶贴装（Eutectic Die Bonding）

用于大功率器件或射频器件：

标准工艺参数：
- 焊料：Au-Si、Au-Sn或Ag-Cu焊料
- 贴装温度：300-450°C（取决于焊料）
- 压力：0.5-5 MPa
- 气氛：还原性（N₂/H₂）或真空
- 共晶比例：Au-20%Sn（熔点280°C）

## 三、引线键合（Wire Bonding）

### 3.1 金丝球焊（Ball Bonding）

使用金线（直径15-50μm）进行球焊：

标准工艺参数（超声波热超声键合）：
| 参数 | 范围 | 典型值 |
|------|------|--------|
| 焊丝直径 | 15-50μm | 25μm（1mil） |
| 超声功率 | 20-100 mW | 40-60 mW |
| 键合时间 | 5-50 ms | 10-20 ms |
| 键合压力 | 20-100 gf | 40-60 gf |
| 热台温度 | 100-200°C | 150°C |
| 尾丝长度 | 0.5-2×线径 | 1×线径 |

球形参数：
- 球径（FAB）：1.5-3×线径（约30-75μm）
- 球高度：0.5-1×线径
- 球剪切力：>30 gf（1-mil Au线）

### 3.2 铝丝楔焊（Wedge Bonding）

使用铝线（直径18-75μm）楔焊：

标准工艺参数：
| 参数 | 范围 | 典型值 |
|------|------|--------|
| 焊丝直径 | 18-75μm | 25-50μm |
| 超声功率 | 20-150 mW | 50-100 mW |
| 键合时间 | 10-100 ms | 30-50 ms |
| 键合压力 | 20-150 gf | 40-80 gf |
| 热台温度 | 150-250°C | 200°C |

### 3.3 键合质量检测

| 测试项目 | 规格 | 方法 |
|---------|------|------|
| 球剪切力 | >30-50 gf | 剪切测试仪 |
| 线拉力（Pull Test） | >5-10 gf | 拉力计 |
| 球形外观 | 无异常变形 | 显微镜 |
| 位置精度 | ±25μm | 测量仪 |

## 四、塑料封装（Plastic Packaging）

### 4.1 转移成型（Transfer Molding）

最常用的塑料封装方式：

标准工艺参数：
- 模具温度：170-200°C
- 注射压力：0.5-15 MPa
- 固化时间：60-180秒
- 合模压力：5-50吨
- 环氧树脂（Molding Compound）：预热50-80°C，粘度控制

工艺流程：
1. 预热芯片框架（Leadframe预热）
2. 合模（Clamping）
3. 注射塑封料（Transfer）
4. 填充（Fill）
5. 固化（Curing）：170-180°C，60-180秒
6. 开模（Opening）

### 4.2 塑封料指标

| 指标 | 规格 |
|------|------|
| 玻璃化转变温度（Tg） | 110-160°C |
| 热膨胀系数（CTE） | 6-15 ppm/°C |
| 吸水率 | <0.5%（ECN） |
| UL阻燃等级 | UL94 V-0 |
| 离子污染（Cl⁻） | <50 ppm |

## 五、先进封装技术

### 5.1 倒装芯片（Flip-Chip）

芯片面朝下通过凸点（Bump）与基板连接：

UBM（Under Bump Metallization）工艺：
- 典型结构：Ti/Cu或Cr/Cu
- 厚度：0.1-1μm
- 工艺：PVD + 电镀

焊球参数：
| 项目 | 规格 |
|------|------|
| 凸点高度 | 50-150μm |
| 凸点直径 | 50-200μm |
| 间距（Pitch） | 150-500μm |
| 凸点材料 | SnAgCu（SAC305最常用） |
| 回流温度 | 230-260°C（峰值） |

底部填充（Underfill）：
- 材料：毛细流动底部填充料（Capillary Underfill）
- 固化：150°C，5-30分钟
- 或NUF（非流动底部填充）
- 填充高度：完全填充芯片与基板间隙

### 5.2 晶圆级封装（WLP）

在晶圆上进行封装后切割：

#### Fan-in WLP
- 凸点制作在芯片表面
- 重布线层（RDL）连接
- 植球后切割

#### Fan-out WLP
- 芯片嵌入塑封料中
- 塑封体表面制作RDL
- 植球切割

### 5.3 3D封装与TSV

TSV（Through-Silicon Via）技术实现芯片堆叠：

TSV工艺参数：
| 工艺 | 参数 |
|------|------|
| 孔径 | 5-50μm |
| 深度 | 50-300μm |
| 刻蚀 | Bosch工艺 DRIE |
| 绝缘层 | SiO₂/SiN，厚度0.5-2μm（ALD或PECVD） |
| 阻挡层/种子层 | Ta/TaN + Cu PVD |
| TSV填充 | 电镀Cu（ECD） |
| 晶圆减薄 | CMP + 磨削，最终厚度50-200μm |
| 露铜（Cu reveal） | CMP或干法刻蚀 |

## 六、测试工艺

### 6.1 CP测试（Chip Probing / Wafer Sort）

在晶圆级对每个芯片进行电性能测试：

测试设备：探针台（Probe Station）+ 测试机（ATE）

标准测试项目：
| 测试项 | 内容 |
|-------|------|
| 开短路测试 | 检查ESD保护二极管导通 |
| 漏电流测试 | Ioff, Ileak |
| 功能测试 | 逻辑功能（Scan/MBIST） |
| DC参数 | IDD, VIL/VIH, VOL/VOH |
| 时序测试 | 频率/时序余量 |

探针卡（Probe Card）：
- 悬臂式探针（Cantilever）：用于高密度测试
- 垂直探针（Vertical Probe）：用于RF/高速测试
- MEMS探针：用于先进制程（<100μmpitch）

探针接触参数：
- 接触力：10-100 gf/针
- 接触电阻：<1Ω
- 超程量（Overdrive）：25-75μm

### 6.2 FT测试（Final Test）

封装后对成品芯片进行最终测试：

标准测试流程：
1. 上料：Tray/Tube/Tape & Reel
2. 测试：与CP类似但使用封装专用接口
3. 分选：OK品→Tape/Reel，Fail品→坏品箱
4. 包装：防静电袋+干燥剂+标签

测试座（Test Socket）：
- 类型：老化测试座、量产测试座
- 接触寿命：10000-100000次
- 寄生参数：<1nH，<1pF（RF应用）

### 6.3 可靠性测试

| 测试项目 | 条件 | 标准 |
|--------|------|------|
| HTOL | 125°C，1000小时 | JESD22-A108 |
| THB | 85°C/85%RH，1000小时 | JESD22-A101 |
| HTSL | 150°C，1000小时 | JESD22-A103 |
| 温度循环（TC） | -65°C~+150°C，500-1000次 | JESD22-A104 |
| 功率循环 | ΔTj=100°C，10000次 | JESD22-A122 |
| 机械冲击 | 1500G，0.5ms | JESD22-B111 |
| 振动 | 20G，20-2000Hz | JESD22-B103 |

## 七、常见问题与解决方案

### 7.1 引线键合强度不足

表现：拉力测试或剪切测试不合格。

原因：
- 键合参数不当（功率/时间/压力）
- 焊丝纯度不足
- 焊盘污染或氧化
- 金属间化合物（IMC）过厚

解决方案：
- 优化键合参数 DOE
- 验证焊丝纯度（>99.99% Au）
- 加强前工序清洗
- 控制焊接后热处理温度和时间

### 7.2 封装分层（Delamination）

表现：塑封料与芯片/框架界面分离。

原因：
- 塑封料固化不足
- 芯片/框架表面污染
- 温度冲击导致CTE失配
- 吸潮后高温回流爆米花效应

解决方案：
- 增加固化时间或温度
- 加强贴装前等离子体清洗
- 使用低CTE塑封料
- 烘烤去潮（125°C，24小时）后再回流

### 7.3 测试良率偏低

原因：
- CP良率问题：工艺波动、颗粒污染
- FT良率问题：封装应力、ESD损伤
- 测试座/探针卡问题

解决方案：
- 排查上游工艺CP良率
- 检查封装操作规范
- 测试设备校准和保养
- 分析Fail模式定位根因

## 九、安全注意事项

1. **化学品安全**：环氧塑封料含活性化学物质，操作需戴手套和口罩，固化时释放小分子气体需通风
2. **高温安全**：模具和烤箱温度高（170-200°C），注意烫伤
3. **静电防护**：所有芯片操作需在防静电工作台上进行，佩戴接地手环
4. **有机溶剂**：IPA、丙酮等易燃有机溶剂远离热源
5. **干燥剂管理**：含干燥剂的包装开封后需在规定时间内使用
6. **洁净室规范**：严格遵守洁净室着装和出入规定，防止颗粒和ESD问题
