# 化学机械抛光（CMP）技术手册

## 一、工艺概述

化学机械抛光（Chemical Mechanical Polishing，简称CMP）是半导体制造中用于实现晶圆表面全局平坦化（Global Planarization）的关键工艺。传统工艺中，薄膜沉积后会形成台阶（Step），影响后续光刻的焦深（Depth of Focus）；CMP通过化学腐蚀和机械研磨的协同作用，可将整个晶圆表面平坦化到纳米级平整度，是先进制程（≤0.35μm）中不可或缺的工艺。

CMP的核心组成：
- 抛光垫（Polishing Pad）：提供机械研磨作用
- 抛光液（Slurry）：含有化学试剂和磨粒
- 抛光头（Head）：固定并加压晶圆
- 抛光台（Platen）：带动抛光垫旋转

## 二、抛光工艺参数

### 2.1 氧化物CMP（Oxide CMP）

典型应用：层间介质（ILD）平坦化、W插塞平坦化

标准工艺参数：
- 抛光液：SiO₂ slurry（硅溶胶型），pH=10-12
- 磨粒尺寸：50-150 nm（硅溶胶）
- 磨粒浓度：5-15 wt%
- 抛光压力：3-6 psi（20-40 kPa）
- 抛光台转速：30-60 rpm
- 抛光头转速：30-60 rpm
- 抛光时间：30-120秒
- 去除速率：200-500 nm/min
- 去除速率均匀性：<10%（within wafer）

### 2.2 钨CMP（W CMP）

典型应用：W插塞（Via Plug）平坦化

标准工艺参数：
- 抛光液：Al₂O₃或SiO₂基，含氧化剂（Fe(NO₃)₃或H₂O₂）
- pH值：2-4（酸性）或 9-11（碱性）
- 抛光压力：3-6 psi
- 去除速率：200-400 nm/min
- W对TEOS选择比：>50:1
- 终点检测：摩擦系数（COF）或阻抗法

### 2.3 铜CMP（Cu CMP）

典型应用：铜互连层平坦化（最关键的CMP工艺）

标准工艺参数：
- 抛光液：Al₂O₃/SiO₂基，含氧化剂（H₂O₂）和抑制剂（BTA/TTA）
- pH值：6-9
- 抛光压力：2-4 psi（分多步：粗抛→精抛→最终清洗）
- 抛光时间：分步控制（总计2-5分钟）
- 去除速率：200-500 nm/min
- Cu对介质层选择比：>100:1

Cu CMP分步工艺：
1. 粗抛（Bulk Removal）：高压，快速去除Cu，10-30秒
2. 精抛（Overburden Removal）：中压，去除凹陷处残留，30-90秒
3. 最终抛光（F buff）：低压，获得完美表面，10-30秒
4. 清洗（Post-CMP Clean）：去除所有颗粒和污染物

### 2.4 硅片清洗（Post-CMP Cleaning）

CMP后残留物清除是关键步骤：

标准清洗参数：
- DI水清洗：2-5分钟，兆声波（Megasonic）辅助
- NH₄OH/H₂O₂（SC-1）：40-60°C，5-10分钟
- HF dip（稀释）：0.5-2%，室温，10-30秒
- IPA干燥：异丙醇蒸汽，30-60秒
- 离心干燥：2000-5000 rpm，30-60秒

## 三、关键质量指标

### 3.1 表面形貌测量

| 指标 | 规格 | 测量设备 |
|------|------|---------|
| 表面粗糙度（Ra） | <0.5 nm（Cu表面） | AFM |
| 局部平整度（3mm×3mm窗口） | <5 nm | 台阶仪 |
| 全局平整度（20mm×20mm或全片） | <20 nm | 干涉仪 |
| 凹陷（Dishing） | <10-30 nm | 台阶仪 |
| 侵蚀（Erosion） | <10-20 nm | 台阶仪 |

### 3.2 缺陷检测

- 颗粒缺陷：SURFSCAN（KLA）或显微镜
- 划伤（Scratch）：显微镜或AFM
- 腐蚀缺陷：显微镜或SEM

典型缺陷规格：
- 颗粒（>0.2μm）：<10 个/片
- 划伤：不允许
- 腐蚀点：<5 个/片

## 四、常见问题与解决方案

### 4.1 凹陷（Dishing）过大

表现：线条区域下凹，导致后续光刻对焦问题。

原因：
- 抛光时间过长
- 压力过大
- 抛光液选择比不合适

解决方案：
- 优化抛光时间，使用终点检测
- 降低压力
- 更换高选择比抛光液

### 4.2 颗粒污染增加

表现：CMP后片子上颗粒明显增加。

原因：
- 抛光液中磨粒团聚
- 抛光垫老化
- 清洗不充分

解决方案：
- 过滤或更换抛光液
- 更换或修整抛光垫
- 优化清洗工艺

### 4.3 划伤（Scratch）

表现：片子上有线性划痕，是致命缺陷。

原因：
- 抛光垫表面有大颗粒
- 抛光液过滤失效
- 抛光垫崩边

解决方案：
- 定期修整抛光垫
- 检查抛光液过滤器
- 更换新抛光垫

## 五、设备与维护

### 5.1 主要设备类型

| 设备类型 | 典型厂商 | 适用 |
|---------|---------|------|
| 12英寸CMP机台 | Applied Materials (Reflexion), Ebara | 先进制程 |
| 8英寸CMP机台 | Applied Materials, SpeedFam | 成熟制程 |
| 抛光液供液系统 | 国产设备 | - |

### 5.2 日常维护

- 抛光垫修整（Conditioning）：每20-50片
- 抛光垫更换：每200-500片
- 抛光液更换：每天或每批次
- 供液管路清洗：每周
- 设备校准：每月

## 六、安全注意事项

1. **化学安全**：抛光液含强碱/酸，需穿戴防护手套、护目镜
2. **机械安全**：抛光机高速旋转部件需安装防护罩
3. **废液处理**：含磨粒和化学品的废液需单独收集处理
4. **粉尘安全**：干粉抛光液操作需配备防尘面具
5. **通风安全**：设备需配备有效排风系统
